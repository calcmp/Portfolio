    Wanting to get into mobile app work, I decided to port my ReactJS TDEE Calculator across to React Native and release it on the google play store.

    Another meaningful project to me is the use of flashcards. I created a flashcard app in React Native, which utilised local storage and redux for state management to allow users to create and save as many decks of flashcards as they like.

    And where we are at now. Demoed is a React Native app targeted towards musicians to record and share demos for their song projects. Utilising redux for state management. Some of the features include; a fully functional media player, record and save recordings, rename, delete, add custom tags, filter based on tags, share recordings to other apps and many more.
